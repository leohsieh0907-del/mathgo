# 數學通

高中自學 App（108 課綱），單一 HTML 檔、無框架無後端，可安裝為 PWA。

**線上版**：https://leohsieh0907-del.github.io/mathgo/

## 內容

| 科目 | 章節 | 題數 |
|---|---|---|
| 數學 | 高一 11／高二 9／高三 7 章 | 515 |
| 物理 | 高一 3／高二 5／高三 5 章 | 260 |
| 化學 | 高一 1 章 | 20 |

每章包含：課程內容（先圖後公式＋公式卡）、練習題（單選／多選／選填／非選，含步驟式詳解與扣分點提醒）、進階實戰題、重點筆記與速記卡。

### 學測訓練

收錄 111～115 學年度學測數學A 共 5 卷 100 題（大考中心公開釋出之試題與答案），**每題附本專案自行撰寫的步驟詳解**（大考中心僅公布答案，不提供詳解）。四種模式：

- **完整卷** — 照原卷題序與時間作答，繳卷估級分
- **隨機組卷** — 依學測配比跨年度抽題，沒寫過的優先
- **章節特訓** — 只練指定章節的真題
- **錯題重練** — 優先抽曾經答錯的題
- **真題研讀** — 不計時不計分，直接翻閱題目與完整詳解

## 檔案

- `math_desktop_v4.html` — 正本（所有內容都在這一個檔）
- `dist/` — 部署產物（GitHub Pages `gh-pages` 分支的內容）
- `gsat/` — 學測真題附圖（自原卷裁切）

## 技術備忘

- 資料依年級索引：`0` 高一、`1` 高二、`2` 高三
- 多科目靠切換全域資料指標＋localStorage 分桶（`mathgo` / `phygo` / `chemgo`）
- 學測題庫 `GSAT_PAPERS`，詳解變數命名 `GSAT<卷id>_SOL_<題號>`，由泛用 IIFE 自動掛載
- 資料字串內的撇號一律用全形 `′`（半形會截斷字串）
- 改版後需將 `dist/sw.js` 的 `CACHE` 版本號 +1 才會更新客戶端快取
